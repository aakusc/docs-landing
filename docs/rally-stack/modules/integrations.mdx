## Overview

Integrations modules connect Rally Stack apps to external systems. They handle authentication, data sync, webhook processing, and error recovery — so your modules don't have to deal with the details of third-party APIs.

## Architecture

All integration modules follow the same pattern:

```typescript
interface IntegrationConnector {
  connect(credentials: Credentials): Promise<Connection>;
  sync(connection: Connection, options: SyncOptions): Promise<SyncResult>;
  webhook(payload: WebhookPayload): Promise<void>;
  disconnect(connection: Connection): Promise<void>;
}
```

Credentials are stored encrypted in the database. The integration module handles token refresh, rate limiting, and retry logic internally.

## Example: Salesforce Sync

```typescript
import { SalesforceConnector } from '@rally/int-salesforce';

const sf = new SalesforceConnector({
  tenantId,
  credentials: await getCredentials(tenantId, 'salesforce'),
});

// One-time sync
const result = await sf.sync({
  objects: ['Opportunity', 'Account', 'Contact'],
  since: lastSyncDate,
});

// Set up webhook for real-time updates
await sf.webhook({
  events: ['OpportunityUpdated', 'AccountUpdated'],
  callbackUrl: `${process.env.APP_URL}/webhooks/salesforce`,
});
```
