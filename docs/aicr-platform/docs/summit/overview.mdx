# Summit Overview

Summit is the **governance and marketplace layer** of AICR. It manages the full lifecycle of packs from promotion through commercial distribution, and provides enterprise-wide oversight across all tenants.

## Summit vs Edge vs Studio

| | Studio | Edge | Summit |
|--|--------|------|--------|
| **Primary user** | Pack authors | Tenant operators | Platform admins + partners |
| **Focus** | Building | Operating | Governing |
| **Pack state** | In development | Active in tenants | In marketplace |
| **Data scope** | Branch database | Tenant-scoped | Platform-wide |

## Summit Access

Summit is accessible to:
- **Platform admins** — full access to all capabilities
- **Enterprise tenants (Summit tier)** — access to marketplace and governance tools
- **Partners** — access to their own pack submissions and distribution metrics

Access to Summit is controlled by the IDCC (Identity Control Center) and requires Summit-tier entitlement.

## Enterprise Governance

Summit provides the platform's enterprise governance capabilities:
- Cross-tenant compliance reporting via the GOCC
- Platform-wide audit trails
- Partner SLA monitoring
- Revenue and usage analytics across all tenants
