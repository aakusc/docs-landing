# API Overview

The AICR Platform API is a REST API for programmatic access to platform resources. Use it to manage tenants, packs, agents, and work items from external systems.

## Base URL

```
https://aicr.aicoderally.com/api
```

## Authentication

All API requests require authentication. See the [Authentication guide](/api-reference/authentication) for details.

## Endpoints

| Resource | Description |
|----------|-------------|
| [Packs](/api-reference/packs) | List, activate, and manage pack versions |
| [Tenants](/api-reference/tenants) | Create and configure tenant environments |
| [Agents](/api-reference/agents) | Query agent status and submit work items |
| [Work Items](/api-reference/work-items) | Track agent tasks and retrieve results |

## Request Format

All requests use JSON:

```http
POST /api/tenants/tenant-abc/packs
Content-Type: application/json
Authorization: Bearer <api-key>

{
  "packId": "approval-workflows",
  "version": "1.2.0"
}
```

## Response Format

All responses return JSON:

```json
{
  "success": true,
  "data": { ... },
  "meta": {
    "requestId": "req-xyz",
    "timestamp": "2026-02-26T10:00:00Z"
  }
}
```

Errors return a consistent shape:

```json
{
  "success": false,
  "error": {
    "code": "PACK_NOT_FOUND",
    "message": "Pack 'approval-workflows' not found",
    "requestId": "req-xyz"
  }
}
```

## Rate Limits

| Plan | Requests/minute | Requests/day |
|------|----------------|-------------|
| Edge | 60 | 10,000 |
| Summit | 300 | 100,000 |
| Enterprise | Custom | Custom |

Rate limit headers are included on every response:
```
X-RateLimit-Limit: 60
X-RateLimit-Remaining: 58
X-RateLimit-Reset: 1740567660
```
